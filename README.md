# Sweet Scoops - Website Toko Es Krim

Website ini dibuat untuk tugas membuat website digital dengan AI. Tema website adalah **jualan es krim** dan sudah dilengkapi dengan **login admin**.

## Fitur Utama
- Desain menarik dan user-friendly
- Navigasi mudah
- Responsif di desktop dan mobile
- Fitur pencarian produk
- SEO dasar (title, meta description, heading rapi)
- Informasi kontak yang jelas
- Testimoni pelanggan
- Artikel/blog section
- Simulasi analisis kinerja dengan local counter
- FAQ / layanan pelanggan
- **Login admin**
- **Dashboard admin** untuk tambah, edit, dan hapus produk

## Akun Admin
- Username: `admin`
- Password: `admin123`

## Cara Menjalankan di VSCode
### Opsi paling mudah: Live Server
1. Buka folder project ini di VSCode.
2. Install extension **Live Server**.
3. Klik kanan file `index.html`.
4. Pilih **Open with Live Server**.
5. Website akan terbuka di browser.

### File utama
- `index.html` = halaman utama website
- `login.html` = halaman login admin
- `admin.html` = dashboard admin
- `style.css` = styling website
- `script.js` = semua logic website dan admin

## Catatan Penting
Login admin pada project ini menggunakan **localStorage** browser, jadi cocok untuk tugas sekolah, demo, atau presentasi. Karena ini bukan backend sungguhan, data admin dan produk tersimpan lokal di browser.
